Request Revision

Enter the reason for requesting a revision from the applicant.

Reject Application

This will permanently reject the application for Pedro Garcia. The applicant will be notified. This action cannot be undone.

Modal Code Pattern 
<!-- Trigger -->
<button class="btn" onclick="document.getElementById('my-modal').showModal()">Open</button>

<!-- Dialog -->
<dialog id="my-modal" class="modal">
  <div class="modal-box">
    <h3 class="text-lg font-bold">Title</h3>
    <p class="py-4">Content here.</p>
    <div class="modal-action">
      <form method="dialog">
        <button class="btn btn-ghost">Cancel</button>
      </form>
      <button class="btn btn-primary">Confirm</button>
    </div>
  </div>
  <form method="dialog" class="modal-backdrop"><button>close</button></form>
</dialog>
